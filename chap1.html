Introduction.

Définition.

HTML signifie Hyper Text Markup Language.
HTML est un langage de description (ou de formatage) de documents et n'est donc pas un langage de programmation puisqu'il ne permet pas de créer un programme.
Il décrit les structures logiques d'un document dont les éléments sont les textes, les titres, les images, les tableaux, les listes, les liens etc.
HTML est un langage extrêmement tolérant aux pannes. Les instructions incomplètes ou erronées sont tout simplement ignorées par les browsers, mais ne provoquent pas de "plantage".

Bref historique

HTML est un développement du langage SGML (Structured Generalized Markup Language) qui remonte à 1969, année ou IBM a voulu réaliser un standard d'interchangeabilité de document.
L'utilisation du HTML date de 1990 et ne cesse d'évoluer continuellement depuis cette date :

  • HTML 1.0 fin 1990
  • HTML 2.0 fin 1995
  • HTML 3.2 début 1997 (première version réellement standard)
  • HTML 4.0 fin 1997
  • XHTML 1.0 début 2000
  • XHTML 2.0 2006

Cette évolution est liée à celle des navigateurs

  • Mosaic (NCSA)
  • Netscape (Netscape Communications Corporation)
  • Internet Explorer (Microsoft)

Logiciels nécessaires.

  • Un fichier HTML est composé de texte "ASCII" (de texte pur) et peut donc être écrit dans n'importe quel éditeur de texte. Ceci dit, les éditeurs les plus performants vérifient la syntaxe, utilisent des couleurs permettant de trouver les erreurs plus rapidement et facilitent l'insertion de balises et de caractères spéciaux. Leur fonctionnalité "WYSIWYG" (What You See Is What You Get), permet d'avoir un aperçu de la manière dont sera publiée la page sur le Web et constitue donc un atout.
  • La visualisation d'un fichier HTML nécessite l'utilisation d'un navigateur (ou browser), qui interprétera le fichier à sa manière. Ceci explique pourquoi les résultats différeront légèrement en fonction du navigateur et de sa version utilisée, et le seul moyen d'avoir la certitude du résultat final est de tester une page HTML dans chaque browser dont on veut connaître l'interprétation particulière.
